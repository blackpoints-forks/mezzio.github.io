PHP micro framework

Mezzio is a Laminas project, a community-supported, open source continuation of Zend Expressive to simply build web applications and APIs.

Build PHP middleware applications, using the PSR-7 (HTTP Messages) and PSR-15 (HTTP Request Handlers) specifications, using a variety of routing and templating options, and the dependency injection container you prefer.

When building applications with Mezzio, you can make use of the various Laminas components, or any third-party components or middleware. You can pick and choose what you need, as and when you need it. Youâ€™re not bound by many, if any, constraints and design decisions.

Features

PSR-15 Middleware

Create middleware applications, using as many layers as you want, and the architecture your project needs.

PSR-7 HTTP Messages

Built to consume PSR-7!

Routing

Route requests to middleware using the routing library of your choice.

Dependency Injection

Make your code flexible and robust, using the dependency injection container of your choice.

Templating

Create templated responses, using a variety of template engines.

Error Handling

Handle errors gracefully, using templated error pages, whoops, or your own solution!

Available PSR-15 Middlewares

In addition of the Mezzio microframework, Mezzio provide an ecosystem of packages to manage session, authentication, authorization, etc. These are PSR-15 Middlewares and easily intregrated in your application even if you don't use the Mezzio microframwork.

You can get the list of all the middlewares in the documentation page.
